Linux_command

Dec 3rd, 2015 10:13 am

this is about the basic command line used most in a linux system

  1. cd..返回上级目录
  2. rm -r递归删除
  3. top查看进程情况,z查看变化情况，q退出
  4. ps aux查看进程情况
  5. 管道作用“|”将左边的输出结果作为右边的输入，比如ps aux | grep ssh查看ssh进程的情况
  6. ifconfig 当前的网络情况
  7. systemctl restart network 系统控制重启网络
  8. vi模式中，一下Insert进入编辑模式，再摁下Insert进入替代模式
